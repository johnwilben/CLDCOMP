VPC Design Challenge

Scenario: TindahanPH Online Store

TindahanPH is launching an online store. They need:

  • A web server that customers can access from the internet
  • A MySQL database that stores products, orders, and customer data
  • The database must NOT be accessible from the internet
  • The database server needs to download security updates from the internet
  • Only the IT admin should be able to SSH into the web server

Design their AWS network below.

1. VPC Design (10 pts)

The VPC is your private network. Choose the right size for TindahanPH.

2. Subnet Placement (15 pts)

Decide where each resource goes. Public subnets can be reached from the internet. Private subnets cannot.

3. Route Tables (15 pts)

Route tables determine where traffic goes. Each subnet needs the right routes.

4. Security Groups (20 pts)

Security Groups are instance-level firewalls. Only allow what's needed.

5. Network ACLs (10 pts)

NACLs are subnet-level firewalls. They add an extra layer of security.

6. Explain Your Design (30 pts)

In your own words, explain the architecture you designed. Why did you make these choices?
